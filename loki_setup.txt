How I started Loki:
sudo docker run -d --name loki --network loki-net -p 3100:3100 -v "$(pwd)/loki-config.yaml":/etc/loki/local-config.yaml:ro -v "$(pwd)/loki-data":/loki grafana/loki:2.9.0 -config.file=/etc/loki/local-config.yaml

How I configured log forwarding:
Promtail was started to scrape /var/log and Docker containers:
sudo docker run -d --name promtail --network loki-net -v /var/log:/var/log:ro -v /var/run/docker.sock:/var/run/docker.sock:ro -v "$(pwd)/promtail-config.yaml":/etc/promtail/config.yml:ro grafana/promtail:2.9.0 -config.file=/etc/promtail/config.yml

Command used to view logs (instant query):
curl -sG 'http://localhost:3100/loki/api/v1/query' --data-urlencode 'query={container=~".*testlogger.*"}' | jq -r '.data.result[]?.values[] | ("\(. [0] / 1000000 | tonumber | strftime("%Y-%m-%dT%H:%M:%S%z"))\t\(. [1])")'
